Vocabulary and Sentence Review
🔄
เรียงประโยคให้ถูกต้องค่ะ
riang bprà-yòok hâi thùuk-dtɔ̂ɔng kâ
Put the words in the correct order.
Examples
1. อ่าน / ฉัน / ชอบ / นิยายแฟนตาซี
2. คน / เค้ก / เป็น / ไทย
3. ไม่ / ปลาทอด / เขา / กิน
1. àan / chǎn / chɔ̂ɔp / ní-yaai fɛɛn-dtaa-sii
2. khon / kék / bpen / thai
3. mâi / bplaa thɔ̂ɔt / kháo / gin
Answers:
1. ฉันชอบอ่านนิยายแฟนตาซี (chǎn chɔ̂ɔp àan ní-yaai fɛɛn-dtaa-sii)
2. เค้กเป็นคนไทย (kék bpen khon thai)
3. เขาไม่กินปลาทอด (kháo mâi gin bplaa thɔ̂ɔt)
✍️
เติมคำในช่องว่างค่ะ

Fill in the blank.
Examples
1. ฉัน ___ ภาษาไทยทุกวัน
2. คุณ ___ ไหมว่านี่คืออะไร
3. อาหารนี้ ___ มาก
1. chǎn ___ phaa-sǎa thai túk wan
2. khun ___ mái wâa nîi khʉʉ à-rai
3. aa-hǎan níi ___ mâak
Possible answers:
1. เรียน (rian) / อ่าน (àan) / พูด (phûut)
2. เข้าใจ (khâo jai) / รู้ (rúu)
3. อร่อย (à-rɔ̀ɔi)

Examples

🧐
ประโยคไหนผิดคะ ลองแก้ให้หน่อยค่ะ
bprà-yòok nǎi phìt ká. lɔɔng gɛ̂ɛ hâi nɔ̀ɔi kâ
Which sentence is wrong? Please try to correct it.
Examples
1. ปลาทอดนี้มากหวาน
2. ฉันหิวข้าว แต่แล้วฉันอิ่ม
1. bplaa thɔ̂ɔt níi mâak wǎan
2. chǎn hǐw khâao dtɛ̀ɛ chǎn lɛ́ɛo ìm
Corrections:
1. ปลาทอดนี้อร่อยมาก (bplaa thɔ̂ɔt níi à-rɔ̀ɔi mâak) - Fried fish is usually delicious/salty, not sweet.
2. ฉันหิวข้าว (chǎn hǐw khâao) OR ฉันอิ่มแล้ว (chǎn ìm lɛ́ɛo) - You can't be hungry and full at the same time.

คุยเรื่องการเรียนภาษาไทย

khui rʉ̂ang gaan rian phaa-sǎa thai
Talking About Learning Thai
🗣️
มาฝึกพูดกันค่ะ: ลองเติมประโยคนี้ให้สมบูรณ์นะคะ
maa fʉ̀k phûut gan kâ: lɔɔng dtəəm bprà-yòok níi hâi sǒm-buun ná ká
Let's practice speaking: Try to complete these sentences.
Examples
1. ฉันเรียนภาษาไทยเพราะ ...
2. ในภาษาไทย ฉันคิดว่าการ ... ยากที่สุด
3. ฉันชอบภาษาไทยเพราะ ...
1. chǎn rian phaa-sǎa thai phrɔ́ ...
2. nai phaa-sǎa thai chǎn khít wâa gaan ... yâak tîi sùt
3. chǎn chɔ̂ɔp phaa-sǎa thai phrɔ́ ...
Example answers:
1. ...ฉันชอบซีรีส์ไทย (...chán chɔ̂ɔp sii-rîi thai)
2. ...ฟัง (...fang) / ...อ่าน (...àan)
3. ...สนุก (...sa-nùk) / ...คนไทยใจดี (...khon thai jai-dii)
💬
ตอบคำถามนะคะ
dtɔ̀ɔp kham-thǎam ná ká
Please answer my questions.
Examples
1. เรียนภาษาไทยนานเท่าไหร่แล้วคะ
2. ชอบเรียนอะไรมากที่สุดคะ พูด ฟัง อ่าน หรือ เขียน
3. ทำไมถึงชอบเรียนอันนั้นคะ
4. ปกติเรียนภาษาไทยที่ไหนคะ
5. เข้าใจที่เค้กพูดวันนี้ไหมคะ
1. rian phaa-sǎa thai naan thâo rài lɛ́ɛo ká
2. chɔ̂ɔp rian à-rai mâak tîi sùt ká. phûut fang àan rʉ̌ʉ khǐan
3. tham-mai thʉ̌ng chɔ̂ɔp rian an nán ká
4. bpà-gà-dtì rian phaa-sǎa thai thîi nǎi ká
5. khâo jai thîi kék phûut wan níi mái ká
1. How long have you been learning Thai?
2. What do you like learning the most? Speaking, listening, reading, or writing?
3. Why do you like learning that one?
4. Where do you usually study Thai?
5. Do you understand what the teacher is saying today?
