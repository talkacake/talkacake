ลัลล์: ลัลล์ไม่ชอบไวน์ค่ะ
lan: lan mâi chôp wai kâ
Lan: Lan doesn't like wine.
Examples
คำศัพท์ (Vocab):
ไม่ชอบ = to dislike
โครงสร้าง (Structure):
ประธาน (Subject) + ไม่ชอบ + กรรม (Object)
(I) + don't like + (coffee).
ฉันไม่ชอบกาแฟ
ตัวอย่าง (Examples):
1. เขาไม่ชอบอากาศร้อน
2. น้องสาวของฉันไม่ชอบหนังผี
ลองแต่งประโยคเองดูนะคะ (Your turn):
... ไม่ชอบ ...
1. kǎo mâi chôp aa-gàat rón
2. nóng-săao kŏng chăn mâi chôp năng pĕe
...
1. He/She doesn't like hot weather.
2. My younger sister doesn't like ghost movies.
Try making your own sentence.

เพื่อน: เกลียดค่ะ อันนี้คือเกลียดมากค่ะ

Examples
คำศัพท์ (Vocab):

2. มาก = very, a lot
Note: 'เกลียด' is much stronger than 'ไม่ชอบ'. The friend is being sarcastic because she thinks Lan secretly likes Wine.

4. A black drink that has caffeine --> It is called...? (Expected: กาแฟ / gaa-fae / coffee)
