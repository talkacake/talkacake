กิจกรรมเสริม

🔍
วิเคราะห์ประโยค: แต่ลืมอะไรไปหรือเปล่า ว่าทำอะไรกับใครไว้บ้าง
wí-krór-bprà-yòhk: dtàe-leum-à-rai-bpai-rĕu-bplàao wâa-tam-à-rai-gàp-krai-wái-bâang
Sentence analysis: But have you forgotten something? What you've done to whom.
Examples
โครงสร้าง '...ไว้' (wái) ใช้บอกว่าทำสิ่งนั้นแล้ว และผลของมันยังอยู่
- ทำการบ้านไว้แล้ว (tam-gaan-bâan-wái-láew) = I've already done the homework (and it's ready).
- ซื้อของมาไว้ให้ (séu-kŏng-maa-wái-hâi) = I bought things for you (and they are here waiting).
ในวิดีโอ 'ทำอะไรกับใครไว้บ้าง' หมายถึง 'the things you have done to people (in the past, and the consequences are still here)'.
ลองแต่งประโยคที่ใช้ '...ไว้' ดูไหมคะ
kroong-sâang ...wái chái-bòk-wâa-tam-sìng-nán-láew láe-pŏn-kŏng-man-yang-yòo
- tam-gaan-bâan-wái-láew = I've already done the homework (and it's ready).
- séu-kŏng-maa-wái-hâi = I bought things for you (and they are here waiting).
nai-wí-dii-oo tam-à-rai-gàp-krai-wái-bâang măai-tĕung 'the things you have done to people (in the past, and the consequences are still here)'.
long-dtàeng-bprà-yòhk-tîi-chái ...wái duu-mái-ká
The structure '...ไว้' (wái) is used to indicate that an action was done and its result remains.
- ทำการบ้านไว้แล้ว = I've already done the homework (and it's ready).
- ซื้อของมาไว้ให้ = I bought things for you (and they are here waiting).
In the video, 'ทำอะไรกับใครไว้บ้าง' means 'the things you have done to people (in the past, and the consequences are still here)'.
