Reviewing Time Vocabulary
✍️
เติมคำในช่องว่าง: ___ คุณจะทำอะไรคะ (เมื่อวานนี้ / พรุ่งนี้)
dtəəm kam nai chɔ̂ng wâang: ___ Khun jà tam à-rai ká (mûea-waan-níi / phrûng-níi)
Fill in the blank: ___ what will you do? (yesterday / tomorrow)
🙋
เลือกคำที่ถูกต้อง: คำไหนแปลว่า 'today' คะ (เมื่อวานนี้ / วันนี้ / พรุ่งนี้)
lʉ̂ak kam tîi tùuk-dtɔ̂ng: kam nǎi bplaae wâa 'today' ká (mûea-waan-níi / wan-níi / phrûng-níi)
Choose the correct word: Which word means 'today'? (yesterday / today / tomorrow)
🔄
เรียงประโยคให้ถูกต้อง: ไป / ที่ไหน / เมื่อวานนี้ / คุณ
riiang bprà-yòok hâi tùuk-dtɔ̂ng: bpai / tîi nǎi / mûea-waan-níi / khun
Put the words in the correct order: go / where / yesterday / you
👍👎
จริงหรือไม่จริงคะ: เราใช้ 'จะ' (jà) พูดเกี่ยวกับ 'เมื่อวานนี้' (mûea-waan-níi)
jing rʉ̌ʉ mâi jing ká: rao chái 'jà' phûut gìao-gàp 'mûea-waan-níi'
True or False: We use 'will' (jà) to talk about 'yesterday'.
🤝
จับคู่ค่ะ: 'เคย' (khəəi) แปลว่าอะไรคะ (already / ever)
jàp-kûu kâ: 'khəəi' bplaae wâa à-rai ká (already / ever)
Matching: What does 'khəəi' mean? (already / ever)

Speaking Practice: Past, Present, Future
🤔
มาช่วยกันคิดกิจกรรมสำหรับแต่ละเวลากันค่ะ

Let's brainstorm some activities for each time frame.
⏪

mûea-waan-níi: gin khâao-pàt, duu nǎng

▶️

โอเคค่ะ ตอนนี้จะถามคำถามนะคะ: เมื่อวานนี้ทำอะไรบ้างคะ

Speaking Practice: Ever or Never
🔁
เรามาฝึกใช้คำว่า 'เคย' (khəəi) และ 'ไม่เคย' (mâi khəəi) กันค่ะ

Let's practice using 'ever' and 'never'.

🎬
