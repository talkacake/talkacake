Grammar: Comparatives

⚖️
Today, we will learn how to compare two things. We call this 'Comparatives'.
วันนี้เราจะเรียนวิธีเปรียบเทียบของสองสิ่งกันค่ะ เราเรียกว่า 'Comparatives' ค่ะ
📖
The simple rule is: (Noun 1) + is/are + (adjective-er / more + adjective) + than + (Noun 2).
กฎง่ายๆ คือ: (ของชิ้นที่ 1) + is/are + (คำคุณศัพท์-er / more + คำคุณศัพท์) + than + (ของชิ้นที่ 2)
Examples
Short adjectives (1-2 syllables): add -er
- cheap -> cheaper
- hot -> hotter
- big -> bigger
Long adjectives (2+ syllables): use 'more'
- delicious -> more delicious
- expensive -> more expensive
- beautiful -> more beautiful
Irregular:
- good -> better
- bad -> worse
คำคุณศัพท์สั้น (1-2 พยางค์): เติม -er
- cheap -> cheaper (ถูกกว่า)
- hot -> hotter (ร้อนกว่า)
- big -> bigger (ใหญ่กว่า)
คำคุณศัพท์ยาว (2+ พยางค์): ใช้ 'more'
- delicious -> more delicious (อร่อยกว่า)
- expensive -> more expensive (แพงกว่า)
- beautiful -> more beautiful (สวยกว่า)
เปลี่ยนรูป:
- good -> better (ดีกว่า)
- bad -> worse (แย่กว่า)
📝
Let's look at some example sentences.
มาดูตัวอย่างประโยคกันค่ะ
Examples
A toastie is **cheaper than** a meal in a restaurant.
Pad Krapow is **more delicious than** a toastie.
7-Eleven is **closer than** the supermarket.
โทสต์ตี้ถูกกว่าอาหารในร้านอาหาร
ผัดกะเพราอร่อยกว่าโทสต์ตี้
เซเว่นอยู่ใกล้กว่าซูเปอร์มาร์เก็ต
❌
Be careful! Don't use 'more' and '-er' together.
ข้อควรระวัง! อย่าใช้ 'more' กับ '-er' ด้วยกันนะคะ
Examples
Mistake: A toastie is **more cheaper** than a burger.
Correct: A toastie is **cheaper** than a burger.
Mistake: Pad Krapow is **deliciouser** than a toastie.
Correct: Pad Krapow is **more delicious** than a toastie.
ผิด: A toastie is **more cheaper** than a burger.
ถูก: A toastie is **cheaper** than a burger.
ผิด: Pad Krapow is **deliciouser** than a toastie.
ถูก: Pad Krapow is **more delicious** than a toastie.
